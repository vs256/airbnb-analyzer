## Commands
help command

## Backend
Build the backend first:
```
make build
```

To run the backend:
```
make run
```

To clean the docker container
```
make clean
```

---

## Frontend
- Make sure React is installed.

Make sure you are in frontend directory
```
cd frontend
```
First time setup will require you to do:
```
npm install
```
To run the reactjs frontend
```
npm start
```
